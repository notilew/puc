JQuery - Animações

Observação: Os parâmetros para os métodos de animação e efeitos podem ser, tempo em milésimos de segundo ou palavras-chave ('fast', 'slow'). Também o tipo de transição, 'linear' ou 'swing'. Também pode-se passar um terceiro parâmetro que é um função que será executada ao término da animação.

Método Descrição
hide() Esconde os elementos selecionados
show() Mostra os elementos selecionados
toggle() Altera a visibilidade dos elementos selecionados

Observação: Os métodos acima alteram altura e opacidade.

JQuery - Efeitos

Método Descrição
fadeOut() Torna os elementos selecionados transparentes
fadeIn() Torna os elementos selecionados opacos
fadeToggle() Inverte a transparência/opacidade dos elementos selecionados
fadeTo() Ajusta a opacidade dos elementos selecionados para um determinado percentual

Observação: Os métodos acima alteram somente a opacidade dos elementos.

JQuery - Efeitos

Método Descrição
slideUp() Reduz a altura dos elementos até escondê-los
slideDown() Restaura a altura original dos elementos
slideToggle() Mostra ou esconde os elementos por meio do ajuste de altura

Observação: Os métodos acima alteram somente a altura dos elementos.

JQuery - Controle de Animação

Método Descrição
animate() Animação personalizada por meio das propriedades CSS
delay() Estabelece uma pausa no intervalo entre duas animações
stop() Interrompe a animação que estiver rodando e passa à próxima da fila
finish() Interrompe todas as animações e aplica a formatação final

Observação: Os métodos acima alteram somente a altura dos elementos.
