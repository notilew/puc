JQuery - Filtros e Processamento

Aprendendo a filtrar e processar elementos selecionados pelo seletor $(), utilizando a biblioteca JQuery

Filtros de Conteúdo

Filtro Descrição
filter() Reduz o conjunto para aqueles que atendem a um seletor ou passem em uma função de teste
is() Testa se pelo menos um objeto atende ao seletor e, nesse caso, retorna verdadeiro. Não retorna outro conjunto
not() Elimina do conjunto os elementos que se enquadrem no segundo seletor
has() Reduz o conjunto àquele elementos que possuam um determinado descendente
map() Transforma o conjunto em um outro objeto contendo novos valores

Filtros de Hierarquia

Filtro Descrição
parent() Retorna os pais de cada elemento selecionado
parents() Retorna todos os ancestrais de cada elemento
parentsUntil() Retorna os ancestrais de um elemento até que o elemento especificado seja encontrado (esse próprio elemento não é incluído na resposta)
closest() Para cada elemento do conjunto, encontra um ancestral que atenda ao seletor (o próprio elemento pode ser retornado se possível)
offsetParent() Retorna o ancestral mais próximo que tenha sido reposicionado no fluxo do documento (por meio da propriedade position da CSS)
children() Retorna os filhos de cada elemento, opcionalmente filtrados por um seletor
find() Retorna os descendentes de cada elemento que atendam a um seletor. É semelhante ao children(), mas o children() retorna apenas os filhos imediatos
siblings() Retorna os irmãos de cada elemento, opcionalmente filtrados por um seletor
next() Retorna o próximo irmão (ou o próximo irmão que atenda a um seletor, se específicado)
nextAll() Retorna os próximos irmãos, opcionalmente filtrados por um seletor
nextUntil() Retorna os próximos irmãos, até que um atenda ao seletor (esse elemento não será incluído)
prev() Retorna o irmão imediatamente anterior
prevAll() Retorna os irmãos anteriores
prevUntil() Retorna os irmãos anteriores, até que um atenda ao seletor (esse elemento não será incluído)

Outros Filtros

Filtro Descrição
add() Adiciona os elementos especificados pelo seletor ao conjunto atual
addBack() Adiciona os últimos elementos filtrados ao conjunto original de elementos
contents() Retorna todos os filhos, incluindo os nodos texto e de comentário
end() Encerra a operação de filtragem e retorna ao conjunto anterior de elementos

Coleções

Método Descrição
get() Retorna um vetor com os elementos selecionados ou apenas um se um índice for especificado
index() Retorna a posição do primeiro elemento do conjunto em relação a seus irmãos no documento. Se um seletor for passado, então o valor de retorno será a posição desse elemento no primeiro conjunto resposta
toArray() Retorna um vetor com os elementos selecionados
