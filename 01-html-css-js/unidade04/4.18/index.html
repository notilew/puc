Manipuladores de Conteúdo

Manipulador Descrição

.html(valor) Define o conteúdo HTML de um elemento
.text() Recupera o conteúdo textual de um elemento
.text(valor) Define o conteúdo textual de um elemento
.val() Recupera o valor do campo de formulário
.val(valor) Define o valor do campo de formulário

Manipuladores de Atributos

Manipulador Descrição
attr(atributo) Recupera um atributo de um elemento
attr(atributo, valor) Define o valor de um atributo de um elemento
attr(objeto) Define os valores dos atributos de um elemento por meio de um objeto
prop(propriedade) Recupera uma propriedade de um elemento
prop(propriedade, valor) Define o valor de uma propriedade de um elemento
prop(objeto) Define os valores das propriedades de um elemento por meio de um objeto

Observação: O manipulador attr() faz referência ao valor da página (valor original após a requisição da página), já o manipulador prop() faz referência ao objeto (document) gerado após a interpretação da página pelo navegador e pode sofres alterações do usuário.

Manipulação de Empacotamento

Manipulador Descrição
wrap(E) Cria um elemento pai E para cada elemento que satifaz o seletor
wrapAll(E) Cria um elemento pai E para o conjunto de elementos que satisfazem o seletor. Se eles estiverem separados, então serão movidos dentro do documento para ficarem agrupados (junto com o primeiro)
wrapInner(E) Envolve o conteúdo de cada elemento que satisfaça ao seletor usando o elemento E
unwrap() Remove o elemento pai, mantendo os elementos selecionados em seus lugares
