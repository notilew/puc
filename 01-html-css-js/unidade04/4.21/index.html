JQuery - Eventos

Aprendendo manipular eventos utilizando a biblioteca JQuery

Propriedades dos Eventos

Propriedade Descrição
type Tipo do evento
target Elemento registrado para o evento
result O valor retornado pelo manipulador do evento
data Dados opcionais passados pelo método do evento
pageX Indica a coordenada X do mouse
pageY Indica a coordenada Y do mouse
preventDefault() Impede que o comportamento padrão seja executado
stopPropagation() Evita que elementos ancestrais executem o evento

Eventos de Clique do Mouse

Evento Descrição
click() Clique do mouse sobre o elemento
dbclick() Duplo clique do mouse sobre o elemento
contextmenu() Clique com o botão direito do mouse
mousedown() Botão do mouse é pressionado sobre o elemento (utilizado quando se arrasta um elemento)
mouseup() Botão do mouse é solto sobre o elemento (utilizado quando se arrasta um elemento)
mouseover() Cursor do mouse entra na área do elemento, excluindo a área dos descendentes
mouseenter() Cursor do mouse entra na área do elemento, incluindo a área dos descendentes (regresso do internet explorer)
mouseout() Cursor do mouse sai da área do elemento, inclusive quando indo para a área dos descendentes
mouseleave() Cursor do mouse sai da área do elemento, independentemente da área dos descendentes (regresso do internet explorer)
mousemove() Cursor do mouse se move sobre o elemento
hover(fn1, fn2) Combinação dos eventos mouseenter() e mouseleave()
Resultado

Eventos do Teclado

Evento Descrição
keypress() Tecla do teclado foi pressionada e solta com o elemento em foco
keydown() Tecla do teclado foi pressionada com o elemento em foco (inclui teclas como shift, esc e control)
keyup() Tecla do teclado foi solta com o elemento em foco
Resultado

Eventos do Formulário

Evento Descrição
focus() O campo do formulário (ou link) recebeu o foco
blur() O campo do formulário (ou link) perdeu o foco
focusin() O elemento ou algum descendente dele recebeu foco
focusout() O elemento ou algum descendente dele perdeu o foco
change() O valor do campo input, textarea ou select foi alterado
select() O usuário fez alguma seleção de texto em um campo input do tipo texto ou em um campo textarea
submit() O usuário tentou enviar o formulário
Resultado

Eventos de Janela (window)

Evento Descrição
resize() A janela do navegador foi redimensionada
scroll() A janela do navegador ou algum elemento com a propriedade overflow definida para scroll, foi rolada
Resultado

Eventos do Documento (document)

Evento Descrição
load() O documento HTML (ou outro recurso específico) foi carregado
ready() Toda a página foi carregada (HTML e recursos)
unload() O usuário saiu da página (navegando outro página ou, em alguns navegadores, fechando a janela)

Eventos do Documento (document)

Evento Descrição
on(evento, função) Define um manipulador de eventos
off(evento) Cancela o manipulador de eventos
one(evento, função) Define o manipulador de evento para uma única execução
trigger(evento) Dispara um evento
